### 🚀 Core memory engine  – [cognee](https://github.com/topoteretes/cognee)  
- **Memory for AI Agents in&nbsp;5 lines of code** – ingest, cognify, query.  
- Graph + Vector fusion for accurate, cost-efficient retrieval.
